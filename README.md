# TFB_Objects

Studying weather objects and total fire ban days in Victoria. Collaborative project with S.Allen

NetCDF files for composite calculation stored in /g/data/gb02/sm5259/TFB_Objects/analysis/composites



This project is structured as following:

```bash
TFB_Objects/
├── analysis
│   ├── example.ipynb
│   └── figures
│       └── .figures
├── .copier-answers.yml
├── data -> /g/data/$PROJECT/$USER/TFB_Objects/data
│   ├── derived
│   └── raw
│       └── DO_NOT_EDIT
├── .envrc
├── .git
├── .gitignore
├── README.md
├── requirements.txt
├── src
│   └── TFB_Objects
│       ├── __init__.py
│       └── sample_fun.py
└── tests
    └── TFB_Objects
        └── test_sample_fun.py
```

The data folder may not be present in the repository but instructions and or documentation are provided.
